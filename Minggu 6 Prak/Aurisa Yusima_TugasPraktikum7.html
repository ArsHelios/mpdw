Analisis Sisaan

Model terbaik hasil identifikasi kemudian dicek asumsi sisaannya. Sisaan model ARIMA harus memenuhi asumsi normalitas, kebebasan sisaan, dan kehomogenan ragam. Diagnostik model dilakukan secara eksplorasi dan uji formal.

Eksplorasi Sisaan

sisaan <- model1$residuals

par(mfrow = c(2,2),    
    mar = c(4,4,2,1),  
    oma = c(0,0,2,0))  

# Q-Q Plot
qqnorm(sisaan, main = "Normal Q-Q Plot")
qqline(sisaan, col = "blue", lwd = 2)

# Plot Sisaan vs Waktu
plot(sisaan, type = "p",
     main = "Plot Sisaan vs Waktu",
     xlab = "Waktu", ylab = "Sisaan")

# ACF
acf(sisaan, main = "ACF Sisaan")

# PACF
pacf(sisaan, main = "PACF Sisaan")

Berdasarkan plot kuantil-kuantil normal, secara eksploratif sisaan dapat dianggap mendekati sebaran normal karena sebagian besar titik-titik cenderung mengikuti garis diagonal, meskipun terdapat beberapa penyimpangan pada bagian ujung sebaran. Selanjutnya, plot sisaan terhadap waktu menunjukkan bahwa sisaan menyebar secara acak di sekitar nilai nol tanpa membentuk pola tertentu, yang mengindikasikan bahwa asumsi kehomogenan ragam telah terpenuhi. Sementara itu, plot ACF dan PACF dari sisaan menunjukkan tidak adanya lag yang signifikan secara statistik, di mana semua spike berada di dalam selang kepercayaan. Hal ini memberikan indikasi kuat bahwa sisaan sudah saling bebas dan tidak mengandung autokorelasi.

Uji Formal

#1) Sisaan Menyebar Normal 
ks.test(sisaan,"pnorm")
## 
##  Asymptotic one-sample Kolmogorov-Smirnov test
## 
## data:  sisaan
## D = 0.13178, p-value = 0.05991
## alternative hypothesis: two-sided

Selain dengan eksplorasi, asumsi tersebut dapat diuji menggunakan uji formal. Pada tahapan ini uji formal yang digunakan untuk normalitas adalah uji Kolmogorov-Smirnov (KS). Hipotesis pada uji KS adalah sebagai berikut.

H0: Sisaan menyebar normal

H1: Sisaan tidak menyebar normal

Berdasarkan uji KS tersebut, didapat p-value sebesar 0.05991 yang lebih besar dari taraf nyata 5% sehingga tak tolak H0 dan menandakan bahwa sisaan menyebar normal. Hal ini sesuai dengan hasil eksplorasi menggunakan plot kuantil-kuantil normal.

#2) Sisaan saling bebas/tidak ada autokorelasi 
Box.test(sisaan, type = "Ljung")
## 
##  Box-Ljung test
## 
## data:  sisaan
## X-squared = 0.0090186, df = 1, p-value = 0.9243

Selanjutnya akan dilakukan uji formal untuk kebebasan sisaan menggunakan uji Ljung-Box. Hipotesis yang digunakan adalah sebagai berikut.

H0: Sisaan saling bebas

H1: Sisaan tidak tidak saling bebas

Berdasarkan uji Ljung-Box tersebut, didapat p-value sebesar 0.9243 yang lebih besar dari taraf nyata 5% sehingga tak tolak H0 dan menandakan bahwa sisaan saling bebas.

#3) Sisaan homogen 
Box.test((sisaan)^2, type = "Ljung")
## 
##  Box-Ljung test
## 
## data:  (sisaan)^2
## X-squared = 25.262, df = 1, p-value = 5.004e-07

Hipotesis yang digunakan untuk uji kehomogenan ragam adalah sebagai berikut.

H0: Ragam sisaan homogen

H1: Ragam sisaan tidak homogen

Berdasarkan uji Ljung-Box terhadap sisaan kuadrat tersebut, didapat p-value sebesar 5.004e-07 yang kurang dari taraf nyata 5% sehingga tolak H0 dan menandakan bahwa ragam sisaan tidak homogen.

#4) Nilai tengah sisaan sama dengan nol 
t.test(sisaan, mu = 0, conf.level = 0.95)

H0: nilai tengah sisaan sama dengan 0

H1: nilai tengah sisaan tidak sama dengan 0
